# My Portfolio SPA

Це односторінковий React-застосунок (SPA), який презентує мене як особистість і початківця-розробника.

## 🔹 Сторінки
- Про мене
- Моє місто
- Мій розвиток

## 🔹 Функціонал
- SPA без перезавантаження сторінок
- маршрутизація (React Router)
- API-запит для отримання погоди
- використання useState та useEffect
- компоненти Header і Footer

## 🔹 Технології
- React
- React Router DOM
- JavaScript
- CSS
- Fetch API
